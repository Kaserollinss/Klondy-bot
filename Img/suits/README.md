- `C.png`
- `D.png`
- `H.png`
- `S.png`

- `/Users/kase/Klondy-bot/Img/Club.png`
- `/Users/kase/Klondy-bot/Img/Diamond.png`
- `/Users/kase/Klondy-bot/Img/Heart.png`
